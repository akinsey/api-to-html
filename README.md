# HOW TO USE

## Configuration

.env

```
EPOCHTALK_PATH='path to epochtalk'
```

## Running

* `npm install`

* `npm run test-env`

    * (outputs your EPOCHTALK_PATH if it is set in .env)

* `npm run gen`

* copy ./doc/api_data.json to ./

* run node index.js

* copy content-output.html and nav-output.html into epochtalk.org site template.
